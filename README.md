# 🍽️ 智能餐厅推荐系统

## 项目简介

智能餐厅推荐系统是一款基于 Python 开发的餐饮推荐应用，旨在根据用户个人偏好、消费水平以及餐厅综合评价，为用户提供个性化餐厅推荐服务。

系统采用 Python + Tkinter 实现可视化交互界面，结合 MySQL 数据库存储用户信息、餐厅信息以及用户评价数据，通过推荐算法对餐厅进行综合评分，实现智能化推荐。

本项目主要解决用户在餐饮选择过程中存在的信息筛选困难问题，提高餐厅选择效率和用户体验。

---

# ✨ 项目功能

## 1. 用户管理

### 用户注册

用户可以注册个人账号，并填写：

- 用户名
- 密码
- 口味偏好
- 消费水平

系统会保存用户画像信息，为后续智能推荐提供数据支持。


### 用户登录

用户通过账号密码登录系统。

登录成功后进入个人功能页面。

---

# 2. 餐厅信息管理

## 添加餐厅

管理员或用户可以添加新的餐厅信息，包括：

- 餐厅名称
- 菜系类型
- 人均价格
- 餐厅评分
- 餐厅地址


## 查看全部餐厅

系统支持展示数据库中的所有餐厅信息。


展示内容：

|字段|说明|
|-|-|
|餐厅名|餐厅名称|
|菜系|餐厅所属类型|
|人均价格|消费水平|
|评分|综合评分|
|地址|位置|

---

# 3. 餐厅搜索

系统支持多条件搜索。

用户可以根据：

- 餐厅名称
- 菜系类型
- 地址信息

快速查询目标餐厅。

---

# 4. 用户评价系统

用户登录后可以发表餐厅评价。

评价内容包括：

- 餐厅名称
- 用户评分
- 评价内容


用户评价数据会影响餐厅综合推荐结果。

---

# 5. 智能餐厅推荐 ⭐

系统根据用户画像进行个性化推荐。

推荐因素包括：

## （1）口味匹配

根据用户注册时选择的口味偏好，与餐厅菜系进行匹配。


## （2）价格匹配

根据用户消费水平：

- 低消费
- 中消费
- 高消费

匹配合适价格范围。


## （3）餐厅评分

结合餐厅基础评分进行计算。


## （4）用户评价

根据用户评论平均分，提高推荐准确性。


最终生成：

- 推荐分
- 推荐等级
